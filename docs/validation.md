# バリデーション



### signup.jsp -> SignUpServlet

| 項目           | 制限                               | 理由                |
| -------------- | ---------------------------------- | ------------------- |
| 出席番号       | 4桁半角数字                        | カラム CHAR(4)      |
| 名前           | 30文字以内                         | カラム NVARCHAR(31) |
| メールアドレス | 60文字以内<br />メールアドレス形式 | カラム VARCHAR(63)  |
| パスワード     | 60文字以内<br />半角英数・半角記号 | カラム VARCHAR(63)  |



### login.jsp -> LoginServlet

| 項目           | 制限                               | 理由                |
| -------------- | ---------------------------------- | ------------------- |
| 出席番号       | 4桁半角数字                        | カラム CHAR(4)      |
| パスワード     | 60文字以内<br />半角英数・半角記号 | カラム VARCHAR(63)  |
